kali下载steam++

Created2024-07-13|Updated2024-07-20|杂物间
|Post Views:

配置环境下载过慢
解决1，加速器：
watt toolkit
点击左上角的火狐浏览器，输入https://steampp.net/ ，进入watt官网

点击gitee下载

下载linux版

下载完成之后点击右上角

打开文件，解压

解压完成后压缩包可以删了，终端打开

命令如下：

    ┌──(kali㉿kali)-[~]
    └─$ ls
    Desktop  Documents  Downloads  Music  Pictures  Public  Templates  Videos
                                                                                
    ┌──(kali㉿kali)-[~]
    └─$ cd Downloads
                                                                                
    ┌──(kali㉿kali)-[~/Downloads]
    └─$ ls
    'Steam _v3.0.0-rc.9_linux_x64'
                                                                                
    ┌──(kali㉿kali)-[~/Downloads]
    └─$ cd Steam\ _v3.0.0-rc.9_linux_x64 
                                                                                
    ┌──(kali㉿kali)-[~/Downloads/Steam _v3.0.0-rc.9_linux_x64]
    └─$ ls
    assemblies  dotnet  Icons  modules  native  script  Steam++.sh
                                                                                
    ┌──(kali㉿kali)-[~/Downloads/Steam _v3.0.0-rc.9_linux_x64]
    └─$ ./Steam++.sh
    创建符号链接 ./dotnet/dotnet 到 ./Steam++
    MESA: error: ZINK: failed to choose pdev
    glx: failed to create drisw screen
    libEGL warning: DRI2: failed to authenticate
    MESA: error: ZINK: failed to choose pdev
    libEGL warning: egl: failed to create dri2 screen
    文件具有执行权限。

输入的命令是└─$之后的内容，如┌──(kali㉿kali)-[~] 就是输入ls，可以自行学习linux的命令
└─$ ls
勾选github加速，开启加速

发现需要手动安装依赖

再开一个终端，输入命令

    sudo apt install libnss3-tools


安装完成之后切回之前的终端，输入ctrl+c，关闭steam++，在输入./Sream++.sh使其运行

这里提示证书不存在，去火狐浏览器配置证书

进入火狐浏览器，找到安全证书

找到steam++中的安全证书,直接搜索SteamToolsCertificate.cer文件或者尝试在我标出的文件目录下寻找

善用搜索

点击import，导入证书

突然发现在recent里面有这个证书，导入


接下来就能愉快的加速了
